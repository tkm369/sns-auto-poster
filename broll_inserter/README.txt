===================================
B-Roll Auto Inserter - 使い方
===================================

【概要】
SRTファイルの字幕テキストから自動でPexelsのフリー素材を検索・ダウンロードし、
Premiere Pro挿入用スクリプト（.jsx）を生成するツールです。

--------------------------------------
■ 初回セットアップ（Pythonから実行する場合）
--------------------------------------

1. Pythonをインストール（https://www.python.org/）
2. コマンドプロンプトで以下を実行：

   cd このフォルダのパス
   pip install -r requirements.txt
   python main.py

--------------------------------------
■ 使い方
--------------------------------------

1. Pexels APIキーを取得する
   → https://www.pexels.com/api/ で無料登録

2. ツールを起動する

3. 各項目を入力：
   - SRT ファイル  : Premiere Proから書き出したSRTファイル
   - 出力フォルダ  : ダウンロードした素材の保存先
   - Pexels APIキー: 取得したAPIキー

4. 「▶ 実行」ボタンをクリック
   → 自動でキーワード抽出 → Pexels検索 → DL が行われます

5. 完了後、出力フォルダに以下が生成されます：
   - broll_001.mp4 などの動画ファイル
   - insert_broll.jsx （Premiere Pro用スクリプト）

--------------------------------------
■ Premiere Proへの取り込み方
--------------------------------------

1. Premiere Pro でプロジェクトを開く
2. メニュー：ファイル > スクリプト > スクリプトを実行
3. insert_broll.jsx を選択して実行
4. V2トラックに自動でBロールが挿入されます

※ V2トラックが存在しない場合はあらかじめ追加しておいてください。

--------------------------------------
■ .exeのビルド方法（外注配布用）
--------------------------------------

build.bat をダブルクリック
→ dist\BRollAutoInserter.exe が生成されます

このexeファイルだけを相手に渡せばOKです。
Python不要で動作します。

--------------------------------------
■ 注意事項
--------------------------------------
- Pexelsの利用規約に従ってください（商用利用可、クレジット推奨）
- 検索結果はキーワードの精度に依存します
- 合わない素材は手動で差し替えてください
