ここにキャラクターの参照音声WAVを置いてください。

ファイル名 = Ren'Py の変数名（キャラの var_name）

例:
  sakura.wav    → 桜井愛 の声
  tsuki.wav     → 月城雪 の声
  narrator.wav  → ナレーターの声

default.wav を置くと、専用ファイルがないキャラに自動的に使われます。

推奨:
  - 10〜30秒程度のクリアな音声
  - ノイズが少ないもの
  - モノラル/ステレオどちらでも可

saved_voices/ フォルダの .wav ファイルをそのまま使えます。
例: ../qwen3_tts/outputs/ 以下の生成済み音声を参照音声として利用できます。
