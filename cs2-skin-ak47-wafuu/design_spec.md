# AK-47 | 桜嵐 (Ouka-ran) - デザイン仕様書

## コンセプト
**スキン名:** AK-47 | 桜嵐 (Ouka-ran / Cherry Blossom Storm)
**テーマ:** 嵐の中を舞う桜をモチーフにした和風デザイン
**レアリティ想定:** Classified (紫)
**仕上げ:** Anodized Multicolored

## デザインレイアウト（各部位）

### バレル（銃身）
- 背景: 深紺グラデーション（先端→根元: `#1A1F3C` → `#2E3560`）
- 模様: 小さな桜の花びらが右から左へ舞い散るパターン（散らし模様）
- 密度: 先端に向かうほど花びらが少なくなる

### レシーバー（本体上部）
- 右側面: 大きな桜の木（枝のシルエット）
- 左側面: 三日月と散る花びら
- 上部: 細かい青海波（せいがいは）パターン（波の幾何学模様）

### ストック（銃床）
- 黒漆塗り風の艶あり仕上げ
- 中央に家紋風の円形紋様（桜の花を5枚で構成）
- 金色のラインでふち取り

### マガジン
- 深紺ベース
- 細い金色の縦ラインを2本

### ハンドガード
- ベース: `#1A1F3C`
- 模様: 霞（かすみ）模様（横に流れる帯状の霞）

## テクスチャ詳細仕様

### カラーマップ (ak47_col.png)
- 解像度: 2048 x 2048px
- 形式: PNG (32bit RGBA)
- レイヤー構成（Photoshop / Krita）:
  1. 下地（ベースカラー）
  2. 青海波パターン（乗算レイヤー）
  3. 桜の木シルエット（通常レイヤー）
  4. 花びら散らし（通常レイヤー、透明度60-80%）
  5. 霞模様（スクリーンレイヤー）
  6. 家紋（通常レイヤー）
  7. 金属パーツのマスク（乗算）

### ノーマルマップ (ak47_nrm.png)
- 形式: PNG (RGB, DirectX法線)
- 凹凸を付ける部分:
  - 桜の花びら（微細な凹凸）
  - 家紋部分（浮き彫り風）
  - 青海波部分（軽い波型エンボス）
  - 金属ネジ・リベット部分（既存UVを参照）

### ラフネスマップ (ak47_rgh.png)
- グレースケール（白=粗い / 黒=滑らか）
- 漆塗り部分: 暗め（`#20`〜`#40`）→ 光沢感
- 金属部分: 中間（`#80`〜`#A0`）
- 花びら部分: やや明るめ（`#A0`〜`#C0`）→ マット感

### メタリックマップ (ak47_mtl.png)
- グレースケール（白=金属 / 黒=非金属）
- 金属フレーム部分: 白（`#F0`〜`#FF`）
- 木材・漆部分: 黒（`#00`〜`#10`）
- 金色の紋様: 白（`#E0`）

### ウェアマスク (ak47_wear.png)
- 傷が付きやすい部分を白く
- エッジ、グリップ周辺、バレル先端を白めに
- 漆塗り中心部分は黒め（傷つきにくい）

## Steam Workshop 投稿手順

1. **Steam Workshop Tools を起動**
   - CS2 ライブラリ → 右クリック → 「ゲームプロパティ」→ DLC タブで確認
   - または Steam → ライブラリ → ツール → "CS2 Workshop Tools"

2. **テクスチャのインポート**
   ```
   tools/source2/content/csgo/
   └── models/weapons/v_models/ak47/
       ├── ak47_col.png
       ├── ak47_nrm.png
       └── ...
   ```

3. **.vmat ファイルの編集**
   - `g_tColor` に ak47_col.png を指定
   - `g_tNormal` に ak47_nrm.png を指定
   - `g_flWearProgress` でウェア表現を設定

4. **プレビュー確認**
   - Workshop Tools 内の 3D ビューで確認

5. **投稿**
   - Workshop Tools → File → Export → Steam Workshop
   - タグ: "Weapon Finish", "AK-47", "Rifle"
